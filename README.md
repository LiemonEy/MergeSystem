# Unity Merge System

A lightweight merge gameplay system created for casual browser/mobile games in Unity.

This repository contains a simplified gameplay-focused version of a production merge mechanic used in a released browser game.

## Features

- Grid-based merge gameplay
- Drag & drop interaction
- Item swapping and upgrading
- Merge validation logic
- Dynamic cell management
- Simple progression scaling
- Gameplay-oriented architecture

## Included Systems

### MergeManager
Handles:
- grid initialization
- active cell management
- item spawning
- gameplay flow control

### MergeCell
Handles:
- drag & drop interaction
- merge/swap logic
- visual state updates
- pointer interaction events

## Tech

- Unity
- C#
- Unity EventSystem interfaces

## Notes

This repository is an isolated gameplay-system showcase extracted from a larger production project.
Some platform-specific systems, UI logic and monetization features were intentionally removed for readability.
